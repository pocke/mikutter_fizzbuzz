mikutter_fizzbuzz
======================
1 2 fizz 4 buzz fizz 7 8 fizz buzz 11 fizz 13 14 fizzbuzz

`` @hoge fizzbuzz 256 `` のようなリプライを受けると、fizzbuzzを返します。

## install
    $ cd ~/.mikutter/plugin/
    $ git clone https://github.com/pocke/mikutter_fizzbuzz.git fizzbuzz
